# [Philips Hue V2 Wallplate Decora Rocker Switch (1, 2, 3, 4)-gang](https://www.printables.com/model/1508001-philips-hue-v2-wallplate-decora-rocker-switch-1-2)

# Abstract

Mount the Philips Hue V2 switch directly on your wall over an existing decora rocker switch and keep using that switch!

# Description

Mount the Philips Hue V2 switch directly over an existing Decora Rocker Switch.

 - 1 gang, 2 gang, 3 gang, and 4 gang models
 - Cutouts keep the rocker switch accessible
 - Center recess for 15x2mm magnets
 - Use your own 15x2mm magnets. Compatible with the screws of your existing wallplate.

*Note*: When mounting on the wall, pay attention and orient the bar recess at the top.
*Note*: This plate is designed for the V2 and [other versions have different dimensions](https://www.reddit.com/r/Hue/comments/1g8xz56/new_v2_dimmer_remotes_have_different_dimensions/)
